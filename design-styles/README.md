# Design Styles

Universal design-token themes. Each style ships a token set (W3C design-tokens JSON), a theme preview, and an export (Tailwind config + CSS variables) so it drops into any stack.

> Scaffolding — folders and specs below; artifacts being filled in.
> House rules: **no purple**, accessible contrast, motion-friendly.

| Style | Mood | Base |
|-------|------|------|
| `sand-terra` | OneWave signature — warm sand and terracotta on near-black | dark |
| `midnight-ink` | High-contrast editorial | dark |
| `nordic-frost` | Cool, minimal, airy | light |
| `mono-brutalist` | Monochrome, bold type, hard edges | light |
| `warm-paper` | Cream, editorial, soft | light |

Each token set is structured for theming (color, type, spacing, radius, motion) and exports cleanly to Tailwind and plain CSS.
