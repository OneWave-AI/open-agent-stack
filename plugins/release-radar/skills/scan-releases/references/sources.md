# Canonical Sources

Use the official release notes, changelog, and deprecation pages for each stack item. The list below covers common vendors; for anything not listed, follow the discovery steps.

## AI model and API vendors

- Anthropic: API release notes, model deprecations page, and pricing/model docs.
- OpenAI: changelog, model deprecations page, and platform docs.
- Google (Gemini / Vertex): release notes and model lifecycle pages.
- Meta (Llama): model card pages and announcement posts.
- Mistral: changelog and models docs.
- Cohere: changelog and deprecations docs.

## Dev tools and SDKs

- Provider SDK repos: GitHub Releases and CHANGELOG files.
- Framework repos (Next.js, React, etc.): GitHub Releases and the official blog.
- Hosting and infra (Vercel, Supabase): changelog pages.

## Discovery steps for missing sources

1. Search for "<tool name> changelog" and "<tool name> release notes" and prefer the official domain.
2. Search for "<model or API name> deprecation" or "end of life" to find lifecycle pages.
3. For open-source tools, check the GitHub repo Releases tab and CHANGELOG.md.
4. Prefer a structured changelog or releases feed over blog posts when both exist.

## Notes

- Always read the live page; do not rely on memory.
