# Channel formats

Per-channel specs the `repurpose` skill targets. Keep every asset publish-ready and on one core message.

## LinkedIn post

- Length: 120-220 words.
- Structure: hook line, white space, value body in short lines, soft CTA.
- First two lines must earn the click before the "see more" fold.
- One link maximum, placed at the end or in the first comment note.

## X / Twitter post

- Length: under 280 characters.
- One idea, one strong hook, no thread dependency.
- Hashtags: zero to two, only if they add reach.

## Thread

- 5-8 numbered parts.
- Part 1: hook that promises a payoff.
- Middle parts: one supporting point each, concrete and self-contained.
- Final part: recap plus a single CTA.
- Number parts as 1/, 2/, and so on.

## Email

- Subject line: under 50 characters, curiosity or benefit driven.
- Preheader: one line that extends the subject, not a repeat.
- Body: 120-180 words, short paragraphs, scannable.
- One CTA, one link.

## Short-form video script

- Runtime target: 30-60 seconds.
- Hook lands in the first 3 seconds.
- 3-4 spoken beats; pair each with an on-screen-text cue.
- Label lines: SPOKEN for narration, ON-SCREEN for text, B-ROLL for visual cues.
- Close with a verbal CTA.

## Voice and safety

- Mirror the source tone, sentence length, and phrasing.
- Pull all facts, stats, and quotes from the source only.
- No emoji. No purple in any color reference, token, or preview.
- No placeholder text in delivered assets.
