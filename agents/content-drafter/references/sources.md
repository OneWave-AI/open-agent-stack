# Source ledger

Record every source you cite here. Open each source with WebFetch before adding it. Do not add a source you have not opened.

## Schema

Add one entry per source using this format:

```
### S<n>
- Claim: <the specific claim this source supports>
- Source: <publication or site name>
- Title: <exact page or article title>
- Author: <author or organization, if available>
- Date: <publication or last-updated date>
- URL: <full URL>
- Quote/figure: <exact figure, date, or short quote used>
- Verified: <yes — opened on this run>
```

## Rules

- Assign IDs sequentially: S1, S2, S3.
- Every inline `[S#]` marker in `draft.md` must resolve to an entry here.
- Every entry here should be referenced at least once in `draft.md`.
- Prefer primary sources and official documentation over secondary summaries.
- Prefer recent sources for time-sensitive claims; note the date so reviewers can judge freshness.
- Never fabricate a URL, quote, figure, author, or date.
